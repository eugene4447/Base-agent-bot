# Base-agent-bot

Base AI News Agent
An autonomous AI-driven news agent designed for the Base Network ecosystem.

Overview
This bot bridges on-chain data with real-world intelligence. It performs automated status checks on a smart contract and uses AI to filter and summarize relevant developments from Google News, delivering concise updates directly to Telegram.

Key Features
On-chain Validation: Logic-gated by contract activity and energy balance checks.

AI Synthesis: Powered by Groq (Llama 3.1) for intelligent news summarization.

Fully Autonomous: Runs on Vercel with scheduled triggers via cron-job.org.

Lightweight: Built with Flask, Web3.py, and BeautifulSoup.
